# Lista-zakupow

1. Metody wstawiania elementów
   
append() – Wstawia treść na samym końcu wewnątrz wybranego elementu (jako ostatnie dziecko).                                                                                                
prepend() – Wstawia treść na samym początku wewnątrz wybranego elementu (jako pierwsze dziecko).

2. Metody usuwania treści                                                                                                                                                                   
remove() – Usuwa cały element ze strony (znikają tagi wraz z zawartością).                                                                                                                  
empty() – Usuwa tylko zawartość elementu (zostawia "puste pudełko", czyli sam znacznik np. ul).

3. Tekst vs Kod HTML                                                                                                                                                                        
text() – Pobiera lub ustawia surowy tekst. Jeśli spróbujesz wpisać tagi (np. b), wyświetlą się one jako zwykłe znaki.                                                                     
html() – Pobiera lub ustawia kod HTML. Pozwala na dynamiczne dodawanie tagów (np. pogrubienia czy linków).
